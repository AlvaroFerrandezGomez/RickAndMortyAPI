# RickAndMortyAPI

Sample iOS application to display Rick And Morty API developed in swift.

### Requirements
 - MVVM
 - Clean architecture
 - Testing
 - Code Organization
 - Code Reusability
 - Coordinator
 - Boxing
 - Dependency injection
 - Search feature
 - Response caching
 - Error handling
 - Cache images from Kingfisher
